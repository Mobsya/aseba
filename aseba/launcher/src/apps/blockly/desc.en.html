Blockly

What is Blockly?

Blockly is a block-based programming language created by Google. The Blockly interface for Thymio will display the generated Aseba code on the side for information. It allows you to take a further step into text programming.

How does it work?

Select different Blockly and/or Thymio blocks and assemble them together. Then, press the play button to send the instructions to Thymio.


More information on www.thymio.org and on developers.google.com/blockly/
