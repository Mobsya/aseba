VPL 3 is a block based programming language that allows to learn the logic behind programming.

  • The simplest way to start programming Thymio
  • Students do not need to know how to read to use VPL
  • Drag and drop icons to code
  • From the first program up to understanding variables and loops
  • No need of an internet connection

Find tips and more information clicking on the help (?) button.

Visit www.thymio.org to get learning material and ideas about how to use Thymio in your classroom.
