Visual Programming Language

What is VPL?

It is a block-based visual programming language. It allows you to program Thymio and to learn about the logic of programming without having to learn a complex text language.

How does it work?

Associate an event block (in orange) with one or several action blocks (in blue) to create a behavior. Then, press the Play button to send the instructions to Thymio.
